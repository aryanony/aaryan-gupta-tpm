# Contributing to Aaryan Gupta's Portfolio

Thank you for your interest! This is a personal portfolio project. If you've found a bug or have a suggestion, feel free to open an issue.

## Reporting Issues

1. Check existing issues first
2. Use a descriptive title
3. Include browser/OS details
4. Provide steps to reproduce

## Development Setup

```bash
git clone https://github.com/aryanony/portfolio.git
cd portfolio
node build.js
npx serve dist
```

## Code Style

- Vanilla HTML/CSS/JS only — no frameworks
- CSS custom properties for all design tokens
- Semantic HTML5 elements
- All content driven from `data/*.json`
